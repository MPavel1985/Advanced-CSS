Это шаблон проекта, его можно использовать для любого нового проекта, удаляя лишнее.
Реализовано:
- package.json
- gumpfile.js с модулями для
  - компиляции scss в css
  - минимизации css, html, js
  - работы с фрагментами html
  - конвертацией картинок в формат avif
  - конвертации шрифтов в woff, woff2
  - watch для обновления изменений в браузере
